# Mari Berbudaya Dengan Social Feed

Installation guide :

## Installation

### Prerequisites

- Install Flutter SDK (stable channel) and add to PATH.
- Install Android Studio (Android SDK & emulator) and/or Xcode (macOS, for iOS builds).
- Install Git.
- Verify with:
  ```
  flutter doctor
  ```

### Clone repository

Replace <repository-url> with the project's repository URL:

```
git clone <repository-url>
```

### Install dependencies

```
flutter pub get
```

### Run the app

- Run on default connected device/emulator:
  ```
  flutter run
  ```
- Run on a specific device:
  ```
  flutter devices
  flutter run -d <device-id>
  ```
